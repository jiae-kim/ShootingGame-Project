- 적용 기능

  - 게임의 배경이 되는 공간 생성

  - canvas로 우주선, 적군, 총알 좌표 생성
  
  - 우주선

    - 우주선이 게임 배경 안에서만 움직이도록 설정

    - 우주선의 속도 조절
    
  - 총알

    - spacebar 누를 시 총알 발사
    
    - 연속해서 총알 발사 가능
    
  - 적군
  
    - 랜덤한 위치에서 내려옴

    - 내려오는 속도 조절
  
    - 생성하는 시간 조절
  
  - 점수 및 게임오버
    
    - 적군에 총알이 닿으면 사라지고 1점 획득
    
    - 적군이 바닥에 닿는 순간 게임오버
